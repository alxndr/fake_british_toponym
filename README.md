Generate names of places that sound sorta British.

## Installation

Add this line to your application's Gemfile:

    gem "fake_british_toponym"

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fake_british_toponym

## Usage

    FakeBritishToponym.new
    # => "Port Aberglentun"

...that's all, folks!
